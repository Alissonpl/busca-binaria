O Gitpod é um Visual Studio (Totalmente identico) só que online, permitindo compartilhar os códigos com os colegas de maneira prática e simples, contendo todas 
as funcionalidade do VSCode convencional.

Somente as APIs são possíveis de mostrar publicamente sem uma conta no Gitpod, então para acessar os arquivos você precisa criar uma conta no site.

Criamos um link de edição do código para que você possa acessar e mexer nele avontade caso necessário, não se preocupe que possuimos backups em nossas máquinas.

As mesmas instruções individuais do Projeto (Sequencial e Binária) aplicam-se da mesma forma ao GP, pois ele possui um terminal integrado para execução de comandos.
A unica diferença é que ao deixar eles online, ele irá mostrar uma janela de Preview com um link (Este será o link de acesso ao servidor) 
No caso da Binaria, este link será usado para as chamadas no Insomnia ou Postman.
